弹琴姿势

坐姿：

坐下弹琴时，上身直立，坐琴凳三分之一，坐在琴键的正中间位置。方便自己可以更好的使用到钢琴所有琴键

手部姿势

  1. 不要晃压手腕：手掌不要下压，也不要过度抬起，正常平放
  2. 抬手和收手：开始时，手轻轻抬起，远离键面；结束时，手正常收回
  3. 连续按键：按下一个键时，其他手指可以抬起；在按下下一个键之后，再松开之前的键，从而形成连续而不松散的发音
  4. 指型：手指直立按键，不要折指，也不要过于内扣，大拇指使用指尖进行弹奏
  5. 手臂：在固定键位弹奏时，注意手部不要晃动。
    只有在真正放松的状态下才能演奏出优美的音色。
